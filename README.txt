###################
SPEAPP (APLIKASI PEMBAYARAN SPP)
###################

Aplikasi pembayaran SPP adalah sebuah aplikasi yang digunakan untuk manajemen pembayaran 
uang SPP, baik negeri maupun swasta. SPP atau Sumbangan Pembinaan Pendidikan adalah iuran rutin 
sekolah yang dilakukan oleh siswa setiap satu bulan sekali.

*******************
Prereisite
*******************

1. Instal XAMPP
2. Instal Visual Studio Code/Sublime Text
3. Web Server

*******************
Cara Login ke Aplikasi
*******************

1. Extrack ZIP SPEAPP, lalu simpan di XAMPP/htdocs
2. Buka XAMPP, lalu start Apache dan MySQL
3. Buka Web server, lalu search http://localhost/speapp/
4. Lalu login, Masukkan Username admin dan Password admin untuk masuk ke halaman admin,
Masukkan Username petugas dan Password petugas untuk masuk ke halaman petugas dan
Masukkan Username NISN siswa dan Password siswa untuk masuk ke halaman siswa
